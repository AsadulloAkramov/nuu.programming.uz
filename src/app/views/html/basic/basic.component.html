HTML Documents

Har qanday HTML dokument < !DOCTYPE html > nomli deklaratsiya bilan boshlanishi shart

HTML dokumentning o'zi esa < html > bilan boshlanadi va < /html > bilan tugaydi

HTML dokumentning ko'rinadigan qismi esa < body > va < /body > teglari orasida joylashadi

<h1> My first heading </h1>

<p> My first paragraph </p>

<h2> Bu heading 2 </h2>

<h3> Bu heading 3 </h3>
