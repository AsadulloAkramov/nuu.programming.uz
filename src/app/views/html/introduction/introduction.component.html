HTML Introduction

HTML web sahifalar yaratish uchun standart markup tildir

HTML nima?

  • HTML to'liq kengaytmasi Hyper Text Markup Language
  • HTML web sahifalar yaratish uchun mo'ljallangan belgilash tilidir
  • HTML web sahifaning strukturasini belgilab beradi
  • HTML bir qanc ha elementlar to'plamidan tashkil topadi
  • HTML bevosita brauzer bilan web sahifaning qanday ko'rsatilishi haqida gaplashadi

<h1> My first heading </h1>

<p> My first paragraph </p>

Namunanining izohi

  • < !DOCTYPE html > documentning HTML5 asosida qurilganini aniqlaydi (HTML5 - eng so'nggi versiyasi)

  • < html> HTML sahifaning asosiy root elementi

  • < head> HTML sahifaning meta informatsiyalardan tashkil topadi

  • < title> HTML sahifaning sarlavhasini aniqlaydi

  • < body> HTML sahifaning asosiy kontenti uning body qismida yoziladi

  • < h1> HTML sahifada mavjud bo'lishi mukin bo'lgan eng katta heading element

  • < p> HTML sahifadagi matnlar aynan paragraph(p) elementi ichida yoziladi

HTML element o'zi nima?
